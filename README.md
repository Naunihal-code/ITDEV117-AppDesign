# ITDEV117-AppDesign
milestone 2 submission including all documents. 
Name of the application
 NUTRI LOG

Description of the problem the app solves
-the app helps track calories and macro nutrients. The app also connects clients with professional trainers to help them acheive their goals. 

Target users or audience
- people new to fitness and meal tracking
- people concerned about nutrition 
- professional coaches trainers
- athletes

Key features of the application
- tracks calories for the day and compares with set goal amount
- tracks macro nutrient intake and compares with target amount
- helps connect users with trainers/ nutritionists
- provides reecipes to help people make diet fun

Overview of how the application works
-Users login or sign up, they can see their progress through numbers and visuals like charts. Users can book appointments with trainers or edit the upcoming appointments. Buy recipes and log what they have eaten so far in the day.
